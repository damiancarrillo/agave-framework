Serves as the main interface to Agave for a webapp developer, with the {@link agave.AgaveFilter AgaveFilter} serving as the Front Controller in a Model-View-Controller architecture, and several annotations to describe how handler methods will field HTTP requests. An example of a handler method is: 

public class FunHandler {

    @HandlesRequestsTo("/haveFun")
    public void haveSomeFun(FunForm funForm) throws ServletException, IOException {
        // wheeee!
    }

}
